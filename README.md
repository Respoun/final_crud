## Run project with Docker

### Requirements:
	Make sure you have:
	- pull the project
	- Docker installed
	- Docker compose installed

### Build and run the project:
	Run the commands:
	- docker-compose up --build

### Open the project :
	Open in your favorite browser:
	- http://localhost:4000

### View logs :
	Open :
	- output.log

###  Routes :
